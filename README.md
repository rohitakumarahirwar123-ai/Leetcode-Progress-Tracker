# LeetCode Progress Tracker

A simple web app built with HTML, CSS, and JavaScript to track LeetCode and DSA progress.

## Why I Built This

While solving DSA problems regularly, I often lost track of how many problems I had solved, which topics I had covered, and whether I was being consistent. I wanted a simple way to see my progress in one place, so I built this project.

## Features

- Add solved problems
- Track topics
- View Easy, Medium, and Hard statistics
- Streak counter
- Progress charts
- Local storage support

## Who Can Benefit?

- Students preparing for coding interviews
- Anyone solving LeetCode regularly
- DSA learners who want to track their progress and stay consistent

## Tech Stack

- HTML
- CSS
- JavaScript
- Local Storage
- Chart.js
